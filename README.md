# Robot Programming

## Description

A Node.js application to control a robot within a defined room.

## Project Structure

robot-programming/
├── src/
│ ├── consts.js
│ ├── index.js
│ ├── robot.js
│ ├── room.js
│ └── utils.js
├── test/
│ ├── robot.test.js
│ └── room.test.js
└── README.md


## Installation
npm install


## Running the Application
Run the application using Node.js:

npm start

## Running Tests
Run the tests using Jest:

npm test
